# UI-Components-KMP Architecture

## Overview

Compose Multiplatform UI component library organized into three areas: theme, animation, and accessibility.

## Dependencies

- Compose Multiplatform (runtime, foundation, material3, ui)
- No external dependencies beyond Compose

## Design Decisions

1. Semantic color tokens over raw color values
2. WCAG contrast ratio calculations for accessibility validation
3. Material Design 3 typography scale
4. Composable animation utilities for consistent motion
