# UI-Components-KMP

Kotlin Multiplatform UI component library for Compose: theme system with WCAG accessibility, animation utilities, and accessibility primitives.

## Features

- **Theme System** - Semantic color tokens, typography, dark/light theme, WCAG contrast validation
- **Animation System** - Timing constants, easing curves, micro-interactions, screen transitions, list animations
- **Accessibility** - WCAG-compliant touch targets, keyboard shortcuts, screen reader utilities, focus management

## Platforms

| Platform | Status |
|----------|--------|
| Android | Supported |
| Desktop (JVM) | Supported |
| iOS | Supported |
| Web (Wasm) | Supported |
